## beginWork

对应如下结构的 ReactElement

```
<A>
<B />
</A>
```

当进入 A 的 beginWork 时，通过对比 B
current fiberNode 与 B reactElement,
生成 B 对应的 wip fiberNode。

在此过程中最多会标记 2 类与「结构变化」相关的 flags：

- Placement：插入：a->ab 移动：abc->bca
- ChildDeletion：删除：ul>li*3 -> ul>li*1
- Update：不包含与「属性变化」相关的 flag：<img title="鸡"/> -> <img title="你太美”/>
